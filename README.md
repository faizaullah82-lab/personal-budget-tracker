# Personal Budget Tracking & Dashboard (Excel)

This project is a multi-sheet personal budgeting system built in Excel. 
It includes automated calculations, KPI metrics, category-based tracking, 
and a dashboard visualizing income, expenses, and savings.

## Features
- Automated monthly budget planning
- Income, expense, and savings categorization
- KPI calculations (savings rate, remaining budget, etc.)
- Dashboard with charts and summaries
- Data validation and structured workflow
